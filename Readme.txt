Приложение "Энциклопедия сказок"
Написано на языке Python. Основной библеотекой является PyQt5.
Все нестандартные библеотеки записаны в requirements.txt
В приложении возможен поиск, добавление и удаление сказок,
а также просомотр дополнительной информации о сказке.
Весь дизайн написан с помощью CSS в data/style/style.css
Все изображения хранятся в data/image
Для удобного запуска приложения имеется ярлык main.exe
Для отслеживания удачных результатов использовалась система контроля версий (Git),
а также созда удаленный репозиторий на GitHub 
(https://github.com/escaleksey/tale_project.git)
